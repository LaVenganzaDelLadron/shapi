## 📖 Overview

`shapi` (Smart Hog API) is a robust backend service designed to power intelligent hog farming operations. This Django-based API provides a structured and efficient way to manage various aspects of a smart farm, including device interactions, feeding schedules, growth monitoring, pen organization, and data recording. It serves as the central nervous system for integrating smart devices and applications, enabling data-driven decisions and automation in modern agriculture.

## ✨ Features

-   🎯 **User Authentication & Authorization**: Secure user management with robust authentication mechanisms.
-   📱 **Smart Device Management**: API endpoints for registering, monitoring, and controlling various smart devices within the farm.
-   🍽️ **Automated Feeding Schedules**: Configure and manage feeding routines for hog pens.
-   📈 **Hog Growth Tracking & Analytics**: Record and analyze growth data for individual hogs or batches.
-   🏡 **Pen Management System**: Organize and track hogs within different pens, including capacity and status.
-   📊 **Comprehensive Data Recording**: Centralized logging and storage for various farm activities and sensor data.
-   📦 **Batch Operations**: Support for managing groups of hogs or devices efficiently.
